# Формат RAW / FpaImage

Используется бинарный формат `FpaImage`, совместимый с исходным C++ кодом.

Структура файла:
```
float   tAdc          (4 байта)
int32   cols          (4 байта)
int32   rows          (4 байта)
int32   type          (4 байта)  // значение 3 для int16
uint64  data_size     (8 байт)   // rows * cols * 2
int16[] data
```

Итого заголовок: **24 байта**.

Чтение RAW:
- Если файл содержит корректный заголовок, читаем его и достаём размеры.
- Иначе пытаемся угадать размеры по размеру файла (640×480×2 и др.).
